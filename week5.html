Week 5

In the fifth week of our journey, we delved into intermediate concepts. Our exploration began with the event loop, delving into the essence of JavaScript's single-threaded nature. Understanding asynchronous programming became a pivotal point in our learning path. Throughout the intermediate JavaScript sessions, we covered numerous topics:

• JS Engine
• Execution Context
• Call Stack
• Memory Heap
• Event Loop
• Callback Queue
• Global Scope
• Function Scope
• Block Scope
• Async & Await
• Class syntax and Class inheritance
• Promises
• Object-Oriented Programming


Reflecting on these concepts, it's amusing to see the array of new knowledge awaiting absorption. Among these, classes stood out as particularly enjoyable. Their definition as "templates for creating objects, encapsulating data and functions for manipulation" sparked creativity during our exercises. Crafting a Tamagotchi class and extending its functionalities served as a hands-on application of our learning. Towards the end of the week, our utilization of the inquirer library for user interactions led to the creation of functional programs, culminating in a fun implementation of a rock-paper-scissors game. It was an engaging experience.

There are definitely grounds for further practice and exploration. While I've coded along to videos and referred to documentation, the realisation has dawned that a good level of understanding necessitates consistent application across diverse scenarios. Embracing that I am a beginner and being gentle in my expectations proves to be a valuable lesson along this learning curve.
