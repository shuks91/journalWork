Week 4

Week 4 started with a dive into the Document Object Model (DOM) in JavaScript. This model visualizes HTML documents as a tree, letting us tweak structure, style, and content in real time. We explored an array of JavaScript methods to finesse the DOM for captivating user experiences.

Following our tutor's instructions, I ventured into crafting a key code generator, unknowingly a sneak peek into our week's core project. This involved creating HTML and CSS files, shaping and styling them. My workflow revolved around initially setting up the structure in HTML and utilizing JavaScript's DOM and using keyboardEvent.code property to capture key presses. I ensured adaptability, making the page responsive to varying screen resolutions.

Midweek, I embarked on an adventure to construct a virtual drum kit. This project revolved around DOM intricacies, key events, and jazzing it up with CSS animations. By week's end, I'd woven in event listeners to buttons and keystrokes, linking keyboard taps or button clicks to different drums in a classic drum set. Each click brought forth the distinct beat of the selected drum, making it an interactive experience. I also incorporated a button hover feature to visually indicate the user's interaction, adding an extra layer of engagement to the experience.


In our alternate class, we delved into SQL, focusing on the concept of normalisation. Normalisation serves to minimize redundancy within database tables, streamlining their structure. Our exploration delved deep into the intricacies of First, Second, and Third Normal Form. Initially challenging, I found that hands-on practice, along with comparisons of various tables, helped solidify my understanding.

We also looked into Primary Keys and the pivotal role of Foreign Keys in linking tables. As we progressed, the puzzle pieces started fitting together when we unravelled the different data relationships: One-to-One, One-to-Many, and Many-to-Many. Understanding these relationships helped me figure out how to effectively split up tables.
